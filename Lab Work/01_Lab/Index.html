Questions and answers

  • Q01. What does HTML mean?
    • Answer : HTML, or Hypertext Markup Language, is a markup language for the web that defines the structure of web pages.

  • Q02. what is the difference between HTML and XHTML?
    • Answer : HTML is the standard markup language for creating web pages, while XHTML is a stricter and more standardized version of HTML.

  • Q03. Which do you think is better, HTML or XTHML?
    • Answer :If you're a beginner you should use nothing other than HTML 4. XHTML is a fad that resulted from the XML fervour of 5-10 years ago. XHTML 2 died in fact. XHTML 5 is nothing more than a token gesture to placate those using XML for HTML and not recommended for use by the W3C.

  • Q04. Why do we need to include charset?
    • Answer :The charset attribute specifies the character encoding for the HTML document. The HTML5 specification encourages web developers to use the UTF-8 character set, which covers almost all of the characters and symbols in the world!
